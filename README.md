## User Guide
"Rock, Paper, Scissors" is a simple hand game played between two people. Each player simultaneously forms one of three shapes with their hand: rock (a fist), paper (a flat hand), or scissors (a fist with the index and middle fingers extended). The winner is determined by the rules: rock crushes scissors, scissors cut paper, and paper covers rock. If both players choose the same shape, the game is a tie.

**Choose one of the three options, click on it and wait for the match results !**
